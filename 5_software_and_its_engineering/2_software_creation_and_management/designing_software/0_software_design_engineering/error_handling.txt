Error: Anything that prevents the program from doing what it needs to do 
Types of errors 
    - Bugs
    - Incorrect input
    - Infrastructure problems (e.g. insufficient hard disk)


People interested in errors (Stakeholders that need to be taken into account when
handling errors) : Order of importance
    - End users
    - System Administrators
    - Programmers

Techniques
    - Halt execution
    - Reduced functionality


Exception handling: handling errors locally in code: we need to avoid this (Use Global Exception handling strategies)
